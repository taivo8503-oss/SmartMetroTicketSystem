# Smart Metro Ticket System

## Giới thiệu

Dự án mô phỏng hệ thống bán vé metro thông minh sử dụng thuật toán Dijkstra để tìm đường đi ngắn nhất.

## Chức năng

- Quản lý ga
- Quản lý tuyến
- Tìm đường đi ngắn nhất
- Mua vé
- Thống kê doanh thu
